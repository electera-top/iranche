پرداخت اینترنتی وجه به وسیله درگاه پرداخت واسطه بیت پی (پی لاین)

== Description ==
**Woocommerce BitPay Gateway** این افزونه شما را قادر می سازد تا براحتی اقدام به ایجاد درگاه پرداخت اینترنتی بیت پی (پی لاین) برای پرداخت های محصولات ووکامرس کنید

= امکانات =
 * سازگار با ووکامرس 7 تا 9.4.3 و بالاتر
 * پنل تنظیمات ساده و کاربرپسند
 * تنظیم پیام دلخواه در هنگام پرداخت موفقیت آمیز ، انصراف از پرداخت و یا لغو پرداخت
 * قابلیت نمایش کد رهگیری بیت پی (پی لاین) همراه با کد میانبر
 * نمایش خطاهای درگاه پرداخت
 * قابلیت پرداخت بر اساس واحد پولی ریال و یا تومان و یا هزار تومان و ...
 
== Installation ==
1. افزونه را از طریق منوی 'افزونه ها' در وردپرس آپلود و فعال کنید
2. تنظیمات افزونه را می توانید از طریق قسمت تنظیمات ووکامرس / پیکربندی / پرداخت ها انجام دهید
