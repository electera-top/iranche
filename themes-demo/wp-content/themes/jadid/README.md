# Themee - قالب فروشگاهی وردپرس

قالب فروشگاهی مدرن و ریسپانسیو برای وردپرس با پشتیبانی کامل از WooCommerce.

## ویژگی‌ها

- ✅ طراحی ریسپانسیو و مدرن
- ✅ پشتیبانی کامل از WooCommerce
- ✅ بهینه‌سازی برای موتورهای جستجو (SEO)
- ✅ سازگار با آخرین نسخه وردپرس
- ✅ پشتیبانی از زبان فارسی (RTL)
- ✅ طراحی با Tailwind CSS
- ✅ سرعت بالا و بهینه‌سازی شده

## نصب

1. فایل‌های قالب را در پوشه `/wp-content/themes/themee/` آپلود کنید
2. از پنل مدیریت وردپرس، قالب را فعال کنید
3. پلاگین WooCommerce را نصب و فعال کنید
4. تنظیمات قالب را از بخش "ظاهر > سفارشی‌سازی" انجام دهید

## تنظیمات

### منوها
- منوی اصلی: برای نوار بالای سایت
- منوی موبایل: برای نمایش در دستگاه‌های موبایل

### ویجت‌ها
- سایدبار اصلی
- فوتر

### سفارشی‌سازی
- لوگو سایت
- رنگ‌ها و فونت‌ها
- تنظیمات فوتر

## فایل‌های اصلی

- `style.css` - فایل اصلی قالب
- `functions.php` - توابع قالب
- `index.php` - صفحه اصلی
- `header.php` - هدر سایت
- `footer.php` - فوتر سایت
- `single-product.php` - صفحه محصول
- `archive-product.php` - صفحه فروشگاه

## پشتیبانی

برای پشتیبانی و گزارش باگ، لطفاً با ما تماس بگیرید.
