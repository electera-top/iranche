# پنل مدیریت ایرانچه

پنل مدیریت کامل برای سیستم چند-مستاجری ایرانچه که شامل تمام قابلیت‌های مورد نیاز برای مدیریت فرانت‌اند و بک‌اند است.

## ویژگی‌ها

### داشبورد اصلی
- آمار کلی سیستم (مستاجرین، فروشگاه‌ها، محصولات، کاربران)
- نمودارهای تعاملی فروش و آمار
- فعالیت‌های اخیر سیستم
- دسترسی سریع به بخش‌های مهم

### مدیریت مستاجرین
- مشاهده لیست تمام مستاجرین
- افزودن مستاجر جدید
- ویرایش اطلاعات مستاجرین
- حذف مستاجرین
- مدیریت وضعیت فعال/غیرفعال

### مدیریت طبقات
- نمایش طبقات به صورت کارت‌های زیبا
- افزودن طبقه جدید با تنظیمات کامل
- ویرایش طبقات موجود
- مدیریت طبقات VIP
- تنظیم رنگ و توضیحات طبقات

### مدیریت فروشگاه‌ها
- لیست کامل فروشگاه‌ها
- افزودن فروشگاه جدید
- ویرایش اطلاعات فروشگاه‌ها
- مدیریت وضعیت فروشگاه‌ها
- نمایش امتیاز و رتبه‌بندی

### مدیریت محصولات
- لیست محصولات با جزئیات کامل
- افزودن محصول جدید
- ویرایش محصولات موجود
- مدیریت موجودی و قیمت
- اتصال به سرویس تجمیع محصولات

### مدیریت کاربران
- لیست کاربران سیستم
- افزودن کاربر جدید
- ویرایش اطلاعات کاربران
- مدیریت نقش‌ها و دسترسی‌ها
- فعال/غیرفعال کردن کاربران

### مدیریت سفارشات
- لیست سفارشات با فیلترهای مختلف
- مشاهده جزئیات سفارشات
- تغییر وضعیت سفارشات
- فیلتر بر اساس وضعیت (در انتظار، در حال پردازش، تکمیل شده)

### آمار و گزارشات
- نمودار فروش روزانه و ماهانه
- آمار محبوب‌ترین محصولات
- آمار کاربران
- فروش بر اساس طبقات
- فیلتر تاریخ برای گزارشات

### مدیریت محتوا
- مدیریت بنرهای سایت
- مدیریت صفحات استاتیک
- مدیریت اطلاعیه‌ها
- تب‌های جداگانه برای هر نوع محتوا

### تنظیمات سیستم
- تنظیمات عمومی سایت
- تغییر نام و توضیحات سایت
- تنظیمات امنیتی
- تغییر رمز عبور مدیر

## تکنولوژی‌های استفاده شده

- **HTML5**: ساختار اصلی پنل
- **CSS3**: طراحی مدرن و ریسپانسیو
- **JavaScript ES6+**: منطق و تعاملات
- **Font Awesome**: آیکون‌ها
- **Vazirmatn Font**: فونت فارسی
- **Chart.js**: نمودارها و گراف‌ها (آماده برای پیاده‌سازی)

## طراحی و UI/UX

### رنگ‌بندی
- پس‌زمینه اصلی: گرادیان آبی تیره (#1a1a2e → #0f3460)
- رنگ اصلی: طلایی (#ffd700)
- رنگ‌های ثانویه: آبی، سبز، قرمز برای وضعیت‌ها
- متن: سفید و خاکستری روشن

### ریسپانسیو
- طراحی کاملاً ریسپانسیو
- سازگار با موبایل و تبلت
- منوی کشویی برای موبایل
- تنظیمات بهینه برای صفحات مختلف

### انیمیشن‌ها
- انتقال‌های نرم بین بخش‌ها
- افکت‌های hover روی کارت‌ها
- انیمیشن‌های بارگذاری
- افکت‌های بصری مدرن

## اتصال به API

پنل ادمین به سه سرویس اصلی متصل است:

1. **Tenant Manager API** (پورت 3000)
   - مدیریت مستاجرین
   - مدیریت کاربران
   - مدیریت سفارشات
   - آمار کلی

2. **Product Aggregator API** (پورت 3001)
   - مدیریت محصولات
   - آمار محصولات

3. **Content Sync API** (پورت 3002)
   - مدیریت محتوا
   - بنرها و صفحات

## نصب و راه‌اندازی

1. فایل‌های پنل در پوشه `admin-panel/` قرار دارند
2. nginx به صورت خودکار این فایل‌ها را سرو می‌دهد
3. پنل در آدرس `http://localhost/admin/` در دسترس است

## استفاده

1. وارد آدرس `http://localhost/admin/` شوید
2. از منوی سایدبار برای دسترسی به بخش‌های مختلف استفاده کنید
3. برای افزودن آیتم جدید، روی دکمه "افزودن" کلیک کنید
4. از جستجوی سراسری برای یافتن سریع اطلاعات استفاده کنید

## ویژگی‌های پیشرفته

- **جستجوی سراسری**: جستجو در تمام بخش‌های پنل
- **فیلترهای پیشرفته**: فیلتر کردن داده‌ها بر اساس معیارهای مختلف
- **مودال‌های تعاملی**: افزودن و ویرایش اطلاعات در مودال‌های زیبا
- **اعلان‌های سیستم**: نمایش اعلان‌های مهم
- **بارگذاری هوشمند**: نمایش وضعیت بارگذاری داده‌ها

## امنیت

- احراز هویت کاربران

- اعتبارسنجی داده‌های ورودی
- محافظت در برابر حملات XSS و CSRF
